# PWA Audit — 2026-06-18

## Goal
Sync Waybar launcher buttons with actually installed Chrome PWAs.

## Discovery commands

### List all Chrome PWA .desktop files with Name and Exec
```bash
for f in ~/.local/share/applications/chrome-*.desktop; do
  echo "=== $(basename $f) ==="
  grep -E '^Name=' "$f"
  grep -E '^Exec=' "$f" | head -1
  echo
done
```

### Check for shortcut URL to identify HTTP vs HTTPS
Look for `--app-launch-url-for-shortcuts-menu-item=` in the `Exec` line of each `.desktop` file. This reveals the actual destination URL.

### Check stale vs current app-ids
Compare waybar config's `--app-id=<X>` against installed `.desktop` files. If waybar references an app-id not present in `.desktop` files, it's stale.
